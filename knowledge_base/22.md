# 🔑 SSH (Port 22)
### ⚡ Quick Wins
- [ ] **Auth Methods:** `ssh -v <IP>` (Check if it accepts passwords or just keys)
- [ ] **Banner:** `nc -vn <IP> 22` (Identify exact OpenSSH version)

### 🔍 Enumeration
- **Username Brute:** `kerbrute` (if AD linked) or `hydra -L users.txt -p password <IP> ssh`
- **Weak Algos:** `nmap --script ssh2-enum-algos <IP>`

### 🚀 Exploitation
- **Key Recovery:** Check web roots or backups for `.ssh/id_rsa`.
- **Default Creds:** Check for `root:root`, `admin:admin`, or `pi:raspberry`.
