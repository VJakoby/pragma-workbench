# 🌐 HTTP / HTTPS

### 1. Enumeration
* [ ] **Directories:** 
```
gobuster dir -u http://<IP> -w common.txt
```
```
ffuf http://<IP>/FUZZ -w /usr/share/dirbuster/common.txt
```
* [ ] **Vhosts:** 
```
ffuf -u http://<IP> -H "Host: FUZZ.domain.com" -w subdomains.txt
```
