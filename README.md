# <h1 align="center">📖🔍 Methodology Dashboard (MMD)</h1>

*Work in progress*
## ✦ What is MMD?
**Methodology Dashboard** is a simple and effective workspace dashboard designed for documentations to navigate and aggregate from multiple sources instantly in one unified interface.

## ✦ What is the goal?
Goal of MMD is to make you spend less time searching and more time executing.

## ✦ Highlights
* ✦ The global search is tightly integrated into MMD and powered by [PKBI](https://github.com/VJakoby/pkbi) which provides the indexing and search capabilites to query both:
*   **Local Knowledge Base**: Directly searches your curated markdown files and tactical guides.
*   **Online Documentation**: Dynamically fetches results from indexed external repositories.

## ✦ Current Requirements
* [PKBI](https://github.com/VJakoby/pkbi) must be running separately on the same machine, by default on port 3002.
