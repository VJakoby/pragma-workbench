# Features



## 🔒 Client-Side Encryption & Privacy
PRAGMA protects sensitive engagements and attachments using a strict zero-knowledge architecture:
* **Military-Grade Cryptography:** Full workbench encryption using AES-256-GCM with keys derived via PBKDF2-SHA-512 (600,000 iterations).
* **True Zero-Knowledge:** Encryption and decryption happen entirely client-side. Your plaintext password never touches the server disk, local storage, or the network.
* **Server-Side Isolation:** The backend only stores encrypted ciphertext.
100% Portable: The encrypted workbench remains completely self-contained. Moving your workspace to another machine is as simple as copying a single file.
